# PRD - Sistema de Ventas de Calzado
## Descripción del Proyecto
Este proyecto consiste en una tienda de calzado nacional e importado, donde los usuarios podran ver diferentes productos, conocer sus precios y realizar compras.

## 2. Objetivo

El objetivo es ofrecer una plataforma facil de usar para la venta de zapatos de diferenetes marcas y estilos.

## 3. Roles

Administrador
Vendedor
Cajero

## 4. Requisitos Funcionales

Registrar ventas
Registrar clientes
Registrar productos
Buscar ventas
Calcular ganancias
ver catalogo de productos


## 5. Tipos de productos

Calzado nacional
Calzado importado
Deportivos
Formales
Casual

## 6. Casos de Uso

Venta al contado
Cliente compra botas y paga completo.

Venta fiada
Cliente compra zapatos y paga después.

## 7. Problema que soluciona

Producto
Cantidad
Precio
Cliente
Tipo de pago
## 8. Criterios de Éxito

Registrar ventas sin errores
Consultar clientes
Calcular ganancias correctamente
